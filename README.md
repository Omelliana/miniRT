# miniRT
The miniRT is my first ray tracer coded in C, normed and humble but functional.

**Usage**

Work from terminal
```bash
./miniRT DESCRIPTION_PATH [FLAG]
```
FLAG:
* ```--save``` | to save image to bmp file

**Example**

`./miniRT scenes/multi_shadow.rt`
